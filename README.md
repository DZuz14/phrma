# Pill Cabinet

Creates a Pill Cabinet where users can view and manage their scripts in.

## Key Features

- **Prescription Management**: View and manage your prescriptions in a visually appealing cabinet interface
- **Smart Filtering**: Filter prescriptions by category and status (active/inactive)
- **Low Quantity Alerts**: Get notified when prescriptions are running low (less than 5 units)
- **Auto-Refill Options**: Enable auto-refill for eligible prescriptions
- **Detailed View**: Access comprehensive prescription details including quantity, refills, and instructions
- **Responsive Design**: Fully responsive interface that works on all device sizes
- **Visual Status Indicators**: Clear visual indicators for prescription status and refill notifications

## Dev

1. Install deps: `npm i` or `pnpm i`
2. Visit: `http://localhost:5173/` (should auto open anyways)
